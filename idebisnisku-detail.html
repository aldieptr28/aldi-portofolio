Inventory Management System - Web Based

Website Ide Bisnis adalah platform yang memungkinkan pengguna untuk menyimpan dan mengelola berbagai ide bisnis mereka dengan mudah. Pengguna dapat menyimpan ide bisnis dengan mengisi berbagai informasi seperti judul ide bisnis, deskripsi, strategi, dan kategori. Setiap ide bisnis juga dilengkapi dengan **to-do list** untuk memudahkan implementasi setiap langkah yang diperlukan dalam mengembangkan ide tersebut.

Fitur-fitur utama dalam aplikasi ini mencakup:

- Mengganti tema tampilan untuk kenyamanan pengguna.

- Ekspor dan impor data ide bisnis untuk memudahkan pengelolaan dan penyimpanan file.

- Pengguna dapat meng-upload foto atau file terkait dengan ide bisnis mereka.

- Penyimpanan data dilakukan menggunakan localStorage, sehingga progres data akan tersimpan dengan aman tanpa khawatir kehilangan informasi.

  • Menggunakan metode SDLC Waterfall
  • Dokumentasi: UI?UX
  • Tools: HTML, CSS, Javascript
