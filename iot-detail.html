Monitoring Pendeteksi kebakaran

Proyek ini adalah pengembangan sistem pendeteksi kebakaran cerdas berbasis IoT yang dirancang untuk memberikan peringatan dini dan pemantauan real-time terhadap potensi bahaya kebakaran. Sistem ini memanfaatkan kombinasi sensor untuk deteksi yang komprehensif dan terintegrasi dengan platform cloud serta aplikasi seluler untuk akses informasi yang mudah.

Cara kerja sensor sebagai berikut

- Sensor Flame, Berfungsi untuk mendeteksi keberadaan api secara langsung melalui radiasi inframerah yang dipancarkan oleh api. Ini memberikan kemampuan deteksi yang cepat terhadap sumber api yang terlihat.

- Sensor MQ-2, Bertugas mendeteksi berbagai jenis gas mudah terbakar, seperti metana, butana, propana, hidrogen, dan asap. Sensor ini sangat penting untuk mendeteksi bahaya kebakaran yang mungkin dimulai dari kebocoran gas atau asap sebelum api terlihat secara langsung.

- Mikrokontroler: (Asumsi: Anda akan menggunakan mikrokontroler seperti ESP32 atau NodeMCU untuk mengelola data dari sensor). Mikrokontroler ini berperan sebagai otak sistem, membaca data dari sensor, memprosesnya, dan mengirimkannya ke database.

Website ini juga dilengkapi dengan bagian Artikel yang menyediakan berbagai informasi dan edukasi seputar kesehatan yang dapat dibaca oleh pengguna.

Sama seperti proyek sebelumnya, website ini dibuat dengan anggaran terbatas dan tanpa hosting pihak ketiga, menjadikannya solusi yang efisien dan mudah diakses.
